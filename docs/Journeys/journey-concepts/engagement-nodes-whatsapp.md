# Overview

This section provides information about how the WhatsApp channel can be used as a channel node in journeys and send WhatsApp messages to your users even when they are outside your app or not on your website. You can also view stats for this node. 

# Create a WhatsApp Engagement Node

To create a WhatsApp engagement node:

1. Drag and drop the **WhatsApp** option under the *Engagements* section. 

<Image title="Select WhatsApp engagement node to create a Journey" alt={406} align="center" width="smart" border={true} src="https://files.readme.io/dfb25d4-Engagement_nodes.png">
  Select Engagement Node
</Image>

2. Enter a name for the WhatsApp node. For instance, "Weekend sale reminder".
3. Select *Service Provider* from the list. Currently, *Nexmo* and *Gupshup* are supported.\
   You may also **Add a new service provider**. For more information, refer to [WhatsApp - Setup](https://docs.clevertap.com/docs/setup-whatsapp).

<Image title="Setup WhatsApp service provider" alt={2548} align="center" width="90% " border={true} src="https://files.readme.io/aea44f5-Whatsapp_engagement_node_.png">
  Setup WhatsApp Service Provider
</Image>

4. To create a WhatsApp message, refer to [Define WhatsApp Message](https://docs.clevertap.com/docs/create-message-whatsapp#define-the-message-content).

# Overview

Based on certain parameters, you can filter journeys to view on the Journey listing page.

## Quick Filters

Quick Filters are available on top for the created journeys.

<Image title="Use quick filters for Journeys" alt={589} align="center" width="smart" border={true} src="https://files.readme.io/1b74fc8-Journey_Quick_Filters.png">
  Quick Filters for Journeys
</Image>

The Quick Filters are:

* **Draft**: Selecting this filter displays all journeys that are in draft status. 
* **Running**: Selecting this filter displays all published and active journeys.
* **Complete**: Selecting this filter displays all finished journeys; either based on time or segment.
* **Stopped**: Selecting this filter displays all stopped journeys. You cannot restart these journeys.
* **Paused**: Selecting this filter displays all paused journeys. You may resume these journeys.

## Custom Filters

To customize journey filters:

1. Click ![Filter](https://files.readme.io/a5bfe04-Journey_Filter_Icon.png) beside the *Search* bar. The **Filter Journeys** box appears.

<Image title="Create a custom filter for Journeys" alt={302} align="center" border={true} src="https://files.readme.io/22fbe47-Journeys_Filters.jpg">
  Custom Filter for Journeys
</Image>

2. Select the field to filter journeys by.

* **Entry Type**: Enables you to filter by:
  * Live
  * Past Behavior
* **Status**: Enables categories same as quick filters.
* **Created by**: Enables you to filter by the journey creator using their email address. 
* **Label**: Enables you to filter journeys by any specific labels that were applied to journeys.

3. Click **Apply**. This displays journeys with your applied filter.
4. Clicking **Save Filter** adds your applied filter to the quick filters on top. Give your saved filter a name.

<Image title="Save a custom filter for Journey" alt={294} align="center" width="px" border={true} src="https://files.readme.io/506a95e-Filters_Save.jpg">
  Save a Custom Filter
</Image>

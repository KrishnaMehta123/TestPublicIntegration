### Q. What is the difference between an IntelliNODE goal and a journey goal?

A. Journey goals exit the users from the journey, whereas IntelliNODE goals do not. IntelliNODE goals help assess which path is more successful.

### Q. Can an IntelliNODE goal and a journey goal be the same?

It is recommended not to use the same goals for IntelliNODE and Journeys because they serve different purposes. IntelliNODE's goal helps us to define which path is getting the better conversion. Journey goals exit the users from the journey once they have finished the desired action.

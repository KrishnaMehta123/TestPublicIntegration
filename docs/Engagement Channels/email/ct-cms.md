# Overview

The CleverTap CMS allows you to create and maintain templates and files in a centralized repository. You can call these templates from the What section in your campaign creation flow. It also allows you to have specialized staff or an agency for template creation; they need not bother with other stages of campaign creation. You can control the access to templates and campaign creation via a [role based access](https://docs.clevertap.com/docs/role-based-access-control).

# Templates

The *Templates* section stores the messaging templates for all your messaging channels. You can save and reuse templates from the templates section.. 

## Template View

Click the templates icon from the left navigation bar. The Templates are displayed in a card view. You can also toggle the view to list the templates, sort,  multi-select or delete templates. To create a new template, click the *Create Template* button.

### Additional Template Options

You can right-click the template from the template gallery for more options.  

\<@placeholder for more options>

* Edit: Click open the template editor. Here, you can make further changes to the template. 
* Clone: Click to create a copy of the template. You can then add further edits to the template.
* Rename: Click to change the name of the template. 
* Add label: Click to add a label. Labeling templates will help you organize and search them easily. 
* Delete: Click to delete the template. A template deleted from the Template gallery will not affect the templates in a running campaign.

### Searching Templates

You can use the various filters to filter templates. 

Navigate to *Content > Templates > Messaging Channel*. 

You can use the following filters:

* Time Period: Search by the time period when the template was created.
* Type: Search by the template type.
* Labels: Search the template by the assigned label. 
* Created by: Search by the person that created the template. 

## Create Templates

Navigate to *Content > Templates > Messaging Channel > Create Template*.

### Email Templates

You add content to pre-built templates or customize and build your templates. After creating an email template, you can use it in the current campaign or save it for future campaigns. For more information, about creating Email Templates, refer to [Email Editor](doc:email-editor-templates).

# Files

You can save and reuse the image assets in the *Files* section. The Files section provides a centralized way of storing, uploading, deleting, and organizing files. 

Navigate to *Content > Templates > Messaging Channel > Files*.

## File View

The image files are displayed in a card view. You can also toggle the view to list, sort, multi-select, or delete files. To upload a new file, click the *Upload File* button.

### Upload Files

1. Navigate to *Content > Templates > Messaging Channel > Files*.
2. Click the Upload Files button. The *Upload to CleverTap* window appears.
3. Drag and drop the files in the window.

### Additional File Options

You can right-click  the template from the template gallery for more options.  

\<@placeholder for more options image>

* Copy link: Click to copy the file link.  
* Rename: Click to change the name of the file. 
* Delete: Click to delete the file. A file deleted from the File section will not affect the files in a running campaign.

### Searching Files

You can use the various filters to filter files. 

Navigate to *Content > Files > Messaging Channel*. 

You can use the following filters:

* Time Period: Search by the time period when the file was created.
* Type: Search by the file type.
* Labels: Search the file by the assigned label. 
* Created by: Search by the person who created the file.

# Viewing Web Exit Intent Campaign Stats

Once the campaign has been sent, you can view the statistics from the dashboard by clicking on **Campaigns**, then selecting the campaign of interest to view its total views, clicks, conversions, and revenue from conversions (if applicable).

<Image title="Check total Views, Clicks, Conversions, CTR, and Conversion Performance." alt={2776} align="center" border={true} src="https://files.readme.io/050a706-Web_Exit_Intent_Stats.png">
  Web Exit Intent Campaign Stats
</Image>

* **Views**: Represents the number of times a Web Exit Popup is viewed
* **Clicks**: Represents the number of times users have clicked on the Web Exit Popup.
* **CTR**:  Represents the ratio of Clicks to Views. (CTR = Clicks/Views \* 100).
* **Trend charts**: Represents trends in data over a specific period of time (for example, daily, weekly, monthly).

Check with your development team that: 

* In-App messaging is enabled in Android. For more information, see [Android In-App Notifications](https://developer.clevertap.com/docs/android#section-in-app-notifications).
* In-App messaging is enabled by default in iOS. For more information for custom In-Apps, see [iOS In-App Notifications](https://developer.clevertap.com/docs/ios#section-in-app-notifications).

This process involves adding mParticle details to the CleverTap dashboard. To add the details to the CleverTap dashboard:  
a. Log in to your CleverTap account.  
b. Navigate to _Settings_ > _Partners_ > _Exports_.

c. Click the **mParticle** icon under the _Unconnected partners_ section. The _MParticle_ page is displayed.  
d. Enter the mParticle details.

[block:image]
{
  "images": [
    {
      "image": [
        "https://files.readme.io/89eba47-Enter_details.png",
        "Enter mParticle Credentials",
        2360

      "align": "center",
      "border": true,
      "caption": "Enter mParticle Credentials"
    }
  ]
}
[/block]



e. Click **Save credentials**. On clicking, the message "Credentials Saved" is displayed.

[block:image]
{
  "images": [
    {
      "image": [
        "https://files.readme.io/aaf2df0-Credentials_saved.png",
        "Update mParticle Credentials",
        2422
      ],
      "align": "center",
      "border": true,
      "caption": "Save mParticle Credentials"
    }
  ]
}
[/block]



You can click **Create Campaign** to set up a new data export or click **Cancel** to return to the _MParticle_ page.  
mParticle is now displayed under Connected partners when you navigate to _Settings_ > _Partners_ > _Exports_.

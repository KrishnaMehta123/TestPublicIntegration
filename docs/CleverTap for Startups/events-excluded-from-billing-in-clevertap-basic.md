# Overview

Some system events are excluded from the MAU and Data Point calculations for CleverTap Essentials users.

# Events Excluded from MAU Calculations

The following is the list of events that are excluded from the MAU calculation:

- AB Experiment Disqualified
- AB Experiment Rendered
- AB Experiment Rolled Out
- AB Experiment Stopped
- App Uninstalled
- Geocluster Entered
- Geocluster Exited
- Identity Error
- Identity Reset
- Identity Set
- Notification Delivered
- Notification Sent
- Notification Viewed
- Partner Sync
- Push Impressions
- Reachable By
- Reply Sent
- Session Concluded
- State Transitioned

# Events Excluded from Data Point Calculations

The following is the list of events that are excluded from the data point calculation:

- AB Experiment Disqualified
- AB Experiment Rendered
- AB Experiment Rolled Out
- AB Experiment Stopped
- App Installed
- App Launched
- App Uninstalled
- App Upgraded
- Geocluster Entered
- Geocluster Exited
- Identity Error
- Identity Reset
- Identity Set
- Notification Clicked
- Notification Delivered
- Notification Replied
- Notification Sent
- Reachable By
- Reply Sent
- Session Concluded
- State Transitioned
- UTM Visited

For more information about each of these events, refer to [System Events](doc:events#system-events) and [Debug Events](doc:events#debug-events).

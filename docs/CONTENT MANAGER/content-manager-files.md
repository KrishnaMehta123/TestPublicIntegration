# Overview

Marketing collaterals extensively use various files to add imagery and communicate brand voice. The _Files_ section provides a centralized way of storing, uploading, deleting, and organizing files. You can save and reuse these assets in the _ Files _ section.

# File Specifications

Refer to the following file specifications before uploading the files:

| File Type | File Format Supported              | Maximum Size |
| :-------- | :--------------------------------- | :----------- |
| Image     | .png, .jpeg, .jpg, .webp, .gif     | 1 MB         |
| Video     | .mp4                               | 50 MB        |
| Document  | .pdf,.doc,.docx, .txt, .json, .ics | 2 MB         |
| Audio     | .mp3                               | 4 MB         |

> ğŸ“˜ File names
> 
> File names can include letters, numbers (0-9), underscores (\_), periods (.), hyphens (-), parenethesis(), and spaces. The name must begin with a letter. Additionally, the file name must be unique.

# Upload Files

Upload files to use later in your messaging.

1. Navigate to _Content_ > _Content Manager_ > _Files_.
2. You can drag and drop the files in the editor or click  **Upload Files**  and select them to upload.

## Filter Files

Filter from the saved files to find the required file quickly. To filter the files:

1. Navigate to  _Content_ > _Content Manager_ > _Files_.
2. Click **Filters** and apply the required criteria.

# Organize Files using Folders

Organizing files using folders helps maintain a structured system, making locating and managing content easy. By grouping related files into designated folders, you improve accessibility and ensure efficient file management. You can organize files in a folder or using nested folders. You can also move or copy files to the required folder. 

## Create File Folders

To create new _File_ folders, follow these steps:

1. Navigate to _Content Manager_ > _Files_ and click **Create Folder**. 
2. Enter a name for your folder and click **Create**.

## Move or Copy Files to Folders

To move or copy files to your folders:

1. Navigate to _Content Manager_ > _Files_. The folders are displayed.
2. Double-click to open a folder. The files are displayed.
3. Click **Selection mode** to select multiple files. 
4. Select the required files and click **Organize**.
5. Click **Copy** or **Move** icon. The list of destination folders is displayed. 
6. Select the required folder and click the **Copy** or **Move Here** button at the bottom of the window. 

The files are copied or moved to the destination folders. 

## Navigate and Search Files in Folders

Your files are saved in the respective folders. To search them in folders:

1. Go to _Content Manager_ > _Files_. The folders are displayed with the path. 
2. Double-click to open a folder and view the files. You can go back and forth by clicking the folder name in the breadcrumb path. 
3. Use the search box to find a specific file or expand the pane to filter with specific criteria.

# FAQs

### What is the maximum upload limit for files per account in CleverTap?

The maximum upload limit for files is based on your subscription plan. You can check the upload limit for your CleverTap account from _Organization_ > _Billing_ > _My plan_ > _Content Manager Usage_.

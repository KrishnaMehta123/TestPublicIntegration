### How Does It Work?

Here is a quick look at how CleverTap's Predictions feature works:

- **Data Collection**: The system gathers data based on user behavior and interactions from integrated sources.
- **Analysis**: The system analyzes this data to identify patterns and trends in user actions.
- **Prediction**: The system forecasts the likelihood of specific actions or events based on the analysis using [XGBoost modeling](https://en.wikipedia.org/wiki/XGBoost).
- **Categorization**: Users are categorized into different likelihood groups (High, Medium, and Low) based on their predicted behavior.
- **Campaign Optimization**: These predictions help make informed decisions, ensuring that the marketing efforts are directed toward the most promising users.

# Glossary

Here are some key terminologies you must understand before using the Predictions feature:

| Element           | Description                                                                                                                  |
| :---------------- | :--------------------------------------------------------------------------------------------------------------------------- |
| Goal Events       | The action that you want the target audience to perform within the prediction period (for example, Charged, Product Viewed). |
| Prediction Period | The time frame within which the user is expected to complete the goal (for example, within 3 days, 7 days).                  |
| Active User Base  | Users who have interacted with your platform in the last 6 months.                                                           |
| Target Segment    | The set of users for which the prediction is created and stored.                                                             |
| User Likelihoods  | The probability that a user will perform a specific action or achieve a particular goal within a given time frame.           |

# Overview

CleverTap supports direct integrations with In-App purchasing platforms, allowing you to seamlessly track and analyze purchase events from your app. These integrations enable real-time revenue monitoring and provide deeper insights into user behavior.

CleverTap currently supports the following partner platforms:

- [Purchasely](doc:purchasely)
- [Qonversion](doc:qonversion)
- [RevenueCat](doc:revenuecat)

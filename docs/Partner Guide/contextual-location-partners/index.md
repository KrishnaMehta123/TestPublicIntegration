# Overview

CleverTap’s location integrations allow you to deliver hyper-personalized experiences based on where your users are or where they’ve been. By combining real-time location data with user behavior, you can trigger relevant campaigns, enhance engagement, and drive higher conversions.

CleverTap currently supports the following partner platforms:

* [AccuWeather](doc:accuweather)
* [OpenWeather](doc:open-weather)
* [Radar](doc:radar)
* [Weather API](doc:weather-api)

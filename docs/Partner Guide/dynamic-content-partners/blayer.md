# Integrate CleverTap with B.Layer

To set up the CleverTap integration with your B.Layer account, perform the following steps:

1. [Create In-App Messages in B.Layer](doc:blayer#create-in-app-messages-in-blayer).
2. [Configure In-App Campaign in CleverTap](doc:blayer#configure-in-app-campaign).

1. Log in to your [B.Layer account](https://app.blayer.phiture.com/users/sign_in) and go to Brand Assets from the left menu (![Edit Icon](https://files.readme.io/3c7ed7b247874aaf3867173af061fdd8591ea64dd0e956b7ba89720f13109b78-ea3493fa-866d-425a-82e7-c718f9488a09.png)).

## Configure In-App Campaign in CleverTap

Set up and personalize your In-App campaigns in CleverTap to engage users effectively. To do so, perform the following steps:

1. Go to *Campaigns* from the CleverTap dashboard. 
2. Click **+ Campaign** and select *In-App Messages* from the dropdown.
3. Click **Go to Editor** under the *What* section.
4. Select the template of your choice. For this example, we are using *Custom HTML Templates*.

> ðŸ“˜ Include Javascript
>
> Ensure you select the *Include JavaScript* checkbox.

<Image alt="Custom HTML Templates" align="center" width="75% " border={true} src="https://files.readme.io/bba3ad2b5f24447d2d31901ac4b17efe32de2cadc35424f7d8ad323841e5869f-image.png">
  Custom HTML Templates
</Image>

5. Delete the default code and paste the HTML snippet copied from B.Layer in *Step 4* of [Create In-App Messages in B.Layer](doc:blayer#create-in-app-messages-in-blayer).

<Image alt="Insert the HTML Code Snippet" align="center" width="75% " border={true} src="https://files.readme.io/b78d02b1fcea9eb477a1945247318bca3c3b8b7bf81ac78888b3f06ad3422a81-Screen_Recording_2025-02-18_at_3.23.24_PM_1.gif">
  Insert HTML Code Snippet
</Image>

6. Add personalization to your In-App message by typing `{{`, `{`, or clicking `@` inside the HTML editor.\
   You can personalize elements such as the message body, button label, or title using [CleverTap Liquid Tags](doc:liquid-tags).

<Image alt="Personalization" align="center" width="75% " border={true} src="https://files.readme.io/34fca332ac701eec9b74a2fca62054ffb8c2f9b30bbdd351abacb56a843eca92-image.png">
  Personalize Email Campaign
</Image>

7. Click **Save** and **Publish** to send your In-App campaign.

Once published, your campaign will display the In-App message created in B.Layer to targeted users based on your segmentation and scheduling settings.

You can also use the same B.Layer design in an Email campaign using the Rich HTML template. For detailed instructions,  refer to [CleverTap Email Campaigns](doc:create-message-email).

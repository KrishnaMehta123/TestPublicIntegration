# Overview

CleverTap integrates with popular CRM platforms, allowing businesses to automate customer data syncing, track sales activities, and trigger personalized engagement campaigns. This integration ensures a seamless data flow between sales and marketing teams, improving customer interactions and retention.

CleverTap currently supports the following partner platforms:

- [Formsuite](doc:formsuite)
- [FreshSales](doc:freshsales)
- [HubSpot](doc:hubspot-via-zapier)
- [Iterate](doc:iterate)
- [Instapage](docs:instapage-via-zapier)
- [LeadSquared](doc:leadsquared)
- [Pipedrive](doc:pipedrive-via-zapier)
- [Salesforce CRM](doc:salesforce-via-zapier)
- [Zoho Form](doc:zoho-form-via-zapier)

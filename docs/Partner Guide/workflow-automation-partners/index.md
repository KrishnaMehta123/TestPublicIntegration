# Overview

CleverTap integrates with leading workflow automation tools to streamline processes and boost productivity. These integrations enable businesses to automate actions across apps, trigger events, and manage data seamlessly, driving greater operational efficiency.

CleverTap currently supports the following partner platforms:

* [Airtable](doc:airtable)
* [Apxor](doc:apxor)
* [DataChannel](doc:datachannel)
* [n8n](doc:n8n)
* [Pipedream](doc:pipedream)
* [SheetDB](doc:sheetdb)
* [Sheetlabs](doc:sheetlabs)
* [Smashpops](doc:smashpops)
* [Zapier](doc:zapier)

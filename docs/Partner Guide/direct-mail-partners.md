# Overview

Enables automated offline engagement through personalized mailers like postcards and brochures. Trigger direct mail campaigns from CleverTap based on user actions or segment criteria.

CleverTap currently supports the following partner platforms:

- [Inkit](doc:inkit)
- [Optilyz](doc:optilyz)
- [Poplar](doc:poplar)

# Supported Features for Smart TV SDKs

This table describes the supported features and engagement channels for each Smart TV SDK.

| Feature/Engagement                | Android TV/Google TV | Apple TV | Samsung TV (Tizen) | LG Web OS TV | Fire TV | Jio STB |
| :-------------------------------- | :------------------- | :------- | :----------------- | :----------- | :------ | :------ |
| **Custom Events**                 | Yes                  | Yes      | Yes (only 7.0)     | Yes          | Yes     | Yes     |
| **Custom Events with Properties** | Yes                  | Yes      | Yes                | Yes          | Yes     | Yes     |
| **onUserLogin**                   | Yes                  | Yes      | Yes                | Yes          | Yes     | Yes     |
| **Add/Modify User Properties**    | Yes                  | Yes      | Yes                | Yes          | Yes     | Yes     |
| **Push (OOTB)**                   | No                   | No       | No                 | No           | No      | No      |
| **Custom Push**                   | Yes                  | No       | No                 | No           | No      | No      |
| **In Apps/Web Pop-Ups**           | Yes                  | No       | Yes (only 7.0)     | Yes          | No      | No      |
| **App Inbox**                     | Yes                  | No       | Yes                | Yes          | Yes     | Yes     |

**Yes**: Supported

**No**: Not Supported

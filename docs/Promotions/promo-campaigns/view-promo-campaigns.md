# Overview

The *Promo Campaigns* list page allows you to view, manage, and track all your promo campaigns. It is accessible from the CleverTap dashboard under *Promotions* > *Promo Campaigns*.

<Image alt="View Promo Campaigns List" align="center" border={true} src="https://files.readme.io/6f437353f34188411ef6f31f204ba9b4808e5169839857f2934be165c7952aee-image.png">
  View Promo Campaigns List
</Image>

The following table provides an overview of key details available on the promo campaigns list page::

<Table>
  <thead>
    <tr>
      <th>
        Column Name
      </th>

      <th>
        Description
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        **Name and ID**
      </td>

      <td>
        Displays the campaign name along with the campaign ID. Clicking on the campaign name opens the campaign details view.
      </td>
    </tr>

    <tr>
      <td>
        **Status**
      </td>

      <td>
        <p>Indicates the current campaign status. Possible values include:</p><ul><li>**Draft**: Campaign is saved but not yet activated</li><li>**Running**: Campaign is live and actively rewarding users.</li><li>**Scheduled**: Campaign is set to start at a future date and time.</li><li> **Stopped**: Campaign has been archived or manually deactivated.</li><li>**Completed**: Campaign has ended based on the scheduled end time.</li><li>**Approval Pending / Rejected**: Campaign is awaiting approval or has been rejected (if applicable).</li></ul>
      </td>
    </tr>

    <tr>
      <td>
        **Campaign Type**
      </td>

      <td>
        Displays whether the campaign is a 

        *Live Behavior Campaign*

         or 

        *Past Behavior Campaign*

        .
      </td>
    </tr>

    <tr>
      <td>
        **Start Date**
      </td>

      <td>
        Displays the scheduled or actual start date and time of the campaign. If unavailable, shows N/A.
      </td>
    </tr>

    <tr>
      <td>
        **Created By**
      </td>

      <td>
        Displays the email ID of the user who created the campaign.
      </td>
    </tr>
  </tbody>
</Table>

# FAQs

Find answers to the following common questions about managing partner vouchers:

### How many voucher codes can I upload per voucher list??

You can upload up to one million codes per voucher list.

### Can I edit a voucher list after it is created?

Yes, you can update the List Name, List Description, and List Expiry.

### What happens if a voucher list expires?

No, expired voucher lists cannot be used in campaigns. However, you can extend the expiry date to make it active again.

### Where can I see the number of issued voucher codes?

Check the *Codes Issued* column for real-time data on distributed voucher codes.

### Can I delete a voucher list linked to an active campaign?

No, you must unlink the voucher list from all campaigns before deleting it.
